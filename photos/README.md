# Model cover photos

Drop JPG photos here. The proposal generator (and contract generator) will look up `photos/{slug}.jpg` for each model. If a photo doesn't exist, the cover falls back to a brand-colored gradient — so this directory can stay partially filled as you go.

## Naming convention

Lowercase, spaces → dashes, `.jpg`. Examples:

- "Big Sky" → `big-sky.jpg`
- "XL 470" → `xl-470.jpg`
- "1-Car" → `1-car.jpg`

## Recommended specs

- 1600 × 1000 px (roughly 16:10), landscape
- JPG format
- ~300–500 KB each (compress before committing)
